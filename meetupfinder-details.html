Meetup Finder - iOS App

Project information

  • Technologies: Swift, Objective-C, UIKit, MapKit, CoreLocation, CocoaPods, MVC Design Pattern, API Networking
  • Year of Release: 2015
  • Github Repository

Meetup Finder is a demo app that helps find when and where the upcoming technology meetups are happening near the userâ€™s current location, using the [meetup.com API]

Features:

  • - Meetup-finder allows users to see the list of upcoming technology meetups around their area by just launching the app
  • - Users can locate and browse the venue locations using the Map View
  • - More detailed information about the event is provided to the users when an event is selected
  • - Users are able to add the event into their iCloud Calendar from the Event Information View
